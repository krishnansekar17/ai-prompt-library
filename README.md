## 🚀 How to Use This Library

1. Browse the folders above
2. Open the folder for the tool you're using
3. Read the `README.md` inside for the exact use case and instructions
4. Copy the `.txt` prompt file and paste it into your AI tool
5. Replace the placeholder variables (shown in `{curly_braces}`) with your actual content
